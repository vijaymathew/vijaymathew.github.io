This post introduces list comprehensions in Clojure, which makes it easy to express many complex operations on lists.

The idea of list comprehensions originates from the comprehensions notation used by mathematicians for deriving new sets from existing sets. For example, the comprehension {x2 | x ∈ {1, 2, 3}} will take one value each from the set {1, 2, 3}, assign that to the variable x and produce a new set with the values of each x squared. In the case the resulting set will be {1, 4, 9}.

The same comprehension can be written in Clojure using the for macro:


((for [x [1 2 3]] (* x x))
;;=> (1 4 9)

Yes, we can also achieve this with map, but the benefit of list comprehensions will apparent as we have to deal with more than one sequence, in more complex ways.
