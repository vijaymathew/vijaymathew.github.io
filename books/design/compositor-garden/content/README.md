# content/

HTML fragments for fragment-based authoring.
One file per chapter or section, without <html>/<head>/<body> tags.

For larger projects, content can be split into:
  00-front-matter.html
  01-on-spacing.html
  02-on-patience.html
  03-the-planting-calendar.html
  04-what-my-teacher-said.html
  05-on-leading.html
  06-the-stone-and-the-page.html
  99-back-matter.html

For this project, all content is in the single index.html file.
This directory is provided for use when splitting into multiple files.
