Scalar Types

Built-in scalar classes: String, Integer, Integer64, Real, Decimal, Boolean, Char.

All scalar classes are modeled as inheriting Any and implementing Comparable and Hashable.

Integer

Bitwise operations use 32-bit integer semantics. Bit index 0 is the least-significant bit. For method calls on integer literals, wrap the literal in parentheses: (5).bitwise_left_shift(1).

Examples

let s := "  Nex  "
print(s.trim().to_upper())        -- "NEX"
print(s.contains("ex"))           -- true

let n: Integer := 7
print(n.plus(5))                  -- 12
print(n.pick())                   -- random integer in [0, 7)
print((5).bitwise_left_shift(1))  -- 10
print((6).bitwise_and(3))         -- 2

let r: Real := 3.6
print(r.round())                  -- 4

let b: Boolean := true
print(b.and(false))               -- false
