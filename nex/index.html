Nex

Web IDE Retired

The browser-based Nex IDE is no longer a supported runtime. It became too costly to maintain reliably alongside the JVM interpreter, Java generator, and JavaScript generator.

The supported workflows are now:

  • the CLI REPL
  • the JVM interpreter
  • generated Java
  • generated JavaScript

For local use, start with the project README and the CLI guide. The language documentation, tutorial, and book remain available.

Nex Home Project README

If a lightweight browser playground returns later, it will be based on a simpler deployment model, not the retired full in-browser IDE/runtime.
