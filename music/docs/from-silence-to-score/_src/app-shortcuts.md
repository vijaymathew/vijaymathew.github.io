Speed in MuseScore comes from the keyboard, not the mouse. You do not need to memorize everything at once — but the handful of shortcuts below, learned a few at a time as you work through the book, will make note entry fluent and keep your hands off the menus. These are the MuseScore 4 defaults. On macOS, read <kbd>Ctrl</kbd> as <kbd>Cmd</kbd> throughout; everything else is the same. You can see and change every shortcut under *Edit ▸ Preferences ▸ Shortcuts*.

## The essential few

If you learn only a dozen shortcuts, learn these — they cover the great majority of what you do while composing.

| Key | Does |
|---|---|
| <kbd>N</kbd> | Enter / leave **note-input mode** |
| <kbd>1</kbd>–<kbd>9</kbd> | Choose a **duration** (see below), then type the note |
| <kbd>A</kbd>–<kbd>G</kbd> | Enter a note of that **letter name** |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Move the selected note **up / down by a semitone** |
| <kbd>Ctrl</kbd>+<kbd>↑</kbd> / <kbd>Ctrl</kbd>+<kbd>↓</kbd> | Move up / down by an **octave** |
| <kbd>Space</kbd> | **Play / stop** playback |
| <kbd>Ctrl</kbd>+<kbd>Z</kbd> | **Undo** (your most important key) |
| <kbd>Esc</kbd> | Leave the current mode / deselect |

## Note input and durations

In note-input mode (<kbd>N</kbd>), first pick a duration, then type a letter to place a note of that length. Rests: pick a duration and press <kbd>0</kbd>.

| Key | Duration |
|---|---|
| <kbd>1</kbd> | sixty-fourth note |
| <kbd>2</kbd> | thirty-second note |
| <kbd>3</kbd> | sixteenth note |
| <kbd>4</kbd> | eighth note |
| <kbd>5</kbd> | quarter note |
| <kbd>6</kbd> | half note |
| <kbd>7</kbd> | whole note |
| <kbd>.</kbd> | make the selected duration **dotted** |
| <kbd>0</kbd> | enter a **rest** of the chosen duration |

| Key | Does |
|---|---|
| <kbd>A</kbd>–<kbd>G</kbd> | enter that note (nearest to the previous one) |
| <kbd>R</kbd> | **repeat** the last note or selection |
| <kbd>Alt</kbd>+<kbd>1</kbd>…<kbd>9</kbd> | add an **interval** above (a 2nd, 3rd, … ) to make a chord |
| <kbd>Backspace</kbd> | delete the last entered note |
| <kbd>Shift</kbd>+<kbd>A</kbd>–<kbd>G</kbd> | add a note of that letter **as a chord** to the current one |

## Pitch, accidentals, and ties

| Key | Does |
|---|---|
| <kbd>↑</kbd> / <kbd>↓</kbd> | raise / lower by a **semitone** |
| <kbd>Ctrl</kbd>+<kbd>↑</kbd> / <kbd>Ctrl</kbd>+<kbd>↓</kbd> | raise / lower by an **octave** |
| <kbd>↑</kbd>/<kbd>↓</kbd> then respell | toggle enharmonic spelling with <kbd>J</kbd> |
| <kbd>T</kbd> | add a **tie** to the next note of the same pitch |
| <kbd>+</kbd> | tie (alternative) |

## Articulations and markings

| Key | Adds |
|---|---|
| <kbd>Shift</kbd>+<kbd>S</kbd> | **staccato** |
| <kbd>Shift</kbd>+<kbd>V</kbd> | **accent** |
| <kbd>Shift</kbd>+<kbd>O</kbd> | **marcato** |
| <kbd>Shift</kbd>+<kbd>N</kbd> | **tenuto** |
| <kbd>S</kbd> | **slur** (extend with <kbd>Shift</kbd>+<kbd>→</kbd>) |
| <kbd>&lt;</kbd> | **crescendo** hairpin over the selection |
| <kbd>&gt;</kbd> | **decrescendo** hairpin |
| <kbd>Ctrl</kbd>+<kbd>K</kbd> | add **lyrics** to the selected note |
| <kbd>Ctrl</kbd>+<kbd>T</kbd> | add **staff text** |
| <kbd>Ctrl</kbd>+<kbd>M</kbd> | add a **rehearsal mark** |

## Selection and editing

| Key | Does |
|---|---|
| <kbd>Shift</kbd>+<kbd>→</kbd> / <kbd>Shift</kbd>+<kbd>←</kbd> | extend the selection **note by note** |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>→</kbd> | extend the selection to the **end of the measure** |
| <kbd>Ctrl</kbd>+<kbd>A</kbd> | select **all** |
| <kbd>Ctrl</kbd>+<kbd>C</kbd> / <kbd>Ctrl</kbd>+<kbd>V</kbd> | **copy / paste** |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>V</kbd> | **selective paste** (pitches only, etc.) |
| <kbd>R</kbd> | **repeat** the selection immediately after it |
| <kbd>Delete</kbd> | remove the selected notes (leaving rests) |

## Navigation and playback

| Key | Does |
|---|---|
| <kbd>Space</kbd> | **play / stop** |
| <kbd>Ctrl</kbd>+<kbd>Home</kbd> | jump to the **start** of the score |
| <kbd>Ctrl</kbd>+<kbd>End</kbd> | jump to the **end** |
| <kbd>→</kbd> / <kbd>←</kbd> | next / previous note or rest |
| <kbd>Ctrl</kbd>+<kbd>→</kbd> / <kbd>Ctrl</kbd>+<kbd>←</kbd> | next / previous **measure** |
| <kbd>Home</kbd> / <kbd>End</kbd> | first / last item in the system |

## Panels

| Key | Opens / toggles |
|---|---|
| <kbd>F6</kbd> | **Properties** panel |
| <kbd>F8</kbd> | **Properties** (some versions) |
| <kbd>F9</kbd> | **Selection filter** |
| <kbd>F10</kbd> | **Mixer** (balance, sounds, mute/solo) |
| <kbd>F11</kbd> | **Play panel** (tempo, volume) |
| <kbd>Ctrl</kbd>+<kbd>S</kbd> | **Save** — do it often |

Learn these gradually. Every "In MuseScore" box in this book leaned on a few of them; by the time you have worked through the chapters, most of this table will already be in your fingers. When you want a shortcut that is not here, open *Edit ▸ Preferences ▸ Shortcuts* and search — and feel free to rebind anything to suit your own hands.
