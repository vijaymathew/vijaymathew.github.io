Slogan is a high-level expression-oriented programming language for general purpose computing.

A Summary of Features

  • Dynamic and strong type system.
  • Rich library: data structures, IO, networking.
  • Unicode support.
  • An efficient multitasking system: execute tens-of-thousands of tasks in parallel.
  • Strong support for declarative programming: higher-order functions, reactive variables, lazy evaluation and pattern matching.
  • Metaprogramming with macros.
  • Modules for large-scale software engineering.
  • REPL for fast prototyping.
  • Compiler for generating high-performance machine binaries.
  • Easy integration with C libraries.

The following code snippet gives a quick taste of what Slogan looks like:

function http_server() 
  handler(open_tcp_server_port(8080));

function handler(server) 
  let (client = read(server)) 
  { display("hello, world", client);
    close_port(client);
    handler(server) };

http_server(); // Starts the HTTP server at localhost:8080.
