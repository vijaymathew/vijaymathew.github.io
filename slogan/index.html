The Slogan Programming Language

Slogan is a high-level, expression oriented programming language that is powerful, easy to learn and fun to use. Slogan is dynamically typed and it elegantly supports a variety of programming styles - including but not limited to - imperative, functional and object oriented. It has efficient high-level data structures and libraries for getting useful stuff done. Slogan supports interactive development with its REPL and interpreter. It has a compiler that generates efficient machine code. There is an API for accessing libraries written in C and a sophisticated macro system for extending the guts of the language.
