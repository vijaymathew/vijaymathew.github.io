The Slogan Programming Language

Slogan is a high-level, expression oriented programming language. Slogan is designed to be powerful, easy to learn and fun to use. It is dynamically typed and elegantly supports a variety of programming styles - including but not limited to - imperative, functional and object oriented. It has efficient high-level data structures and libraries for getting useful stuff done. Slogan supports interactive development with its REPL and interpreter. It has a compiler that generates efficient machine code. There is a versatile API for accessing libraries written in C. Slogan also has a sophisticated macro system that enables you to extend the guts of the language by influencing compile-time code generation.

The following code snippet shows a simple HTTP server written in Slogan. It gives an idea on how programs written in Slogan looks like:

var http_server = function (@optional port = 8080) {
    let server = stream (!tcp_server, [[!port_number port]])
    let handle_client = function (client) {
        println (read_line (client));
        println (to = client, "<html><body><h1>hello from Slogan!</h1></body></html>");
        close_stream (client);
    }
    letrec run_server = function () {
        let client = read (server) {
            task_run (task (function () handle_client (client)));
            run_server ()
        }
    } 
    run_server ()
};

To start the server, call the http_server function from the REPL: 

 > http_server ();

Now if you point your browser to http://localhost:8080 you should see the message "hello from Slogan!"
