Slogan is a powerful programming language for general purpose computing.

A Summary of Features

  • Dynamic and strong type system.
  • Rich library: data structures, IO, networking.
  • Efficient multitasking: execute tens-of-thousands of concurrent tasks.
  • Multi-core parallelism based on actor model.
  • Declarative programming: lazy functions, reactive variables, pattern matching.
  • Meta-programming with macros and compile-time evaluation.
  • Fast prototyping and testing with the interpreter and REPL.
  • A compiler that can generate optimized stand-alone binaries.
  • High-level FFI for integration with C libraries.
  • Unicode support.

The following code snippet will give you a taste of Slogan:

function http_server() 
  handler(tcp_server_stream(8080));

function handler(server) 
  let (client = read(server)) 
   { write("hello, world", client);
     close_stream(client);
     handler(server) };

http_server(); // Starts the HTTP server at localhost:8080.
