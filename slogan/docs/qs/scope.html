Scope, Control, Loops

The let expression is used to bind local variables:

let (a = 34, b = 20) a + b; 
//=> 54

Variable bindings in let will shadow global bindings with the same name:

let a = 10; 
let (a = 34, b = 20) a + b; 
//=> 54

If a binding expression needs to reference the value of a previous binding, letseq should be used:

letseq (a = 34, b = 20 * a) a + b; 
//=> 714

Both the let statement and letseq expression can destructure complex data structures like lists and hashtables.

let [a,b,c] = [1,2,3];
a+b+c;
//=> 6
letseq (#{'name:n, 'age:a} = #{'name:"sam", 'age:30}) [n, a];
//=>[sam, 30]

letrec is used to define recursive local functions:

letrec (a = function(x) if (is_zero(x)) b(x) else a(x - 1), 
        b = function(y) if (is_zero(y)) 'ok else 'blah) 
  a(10); 
//=> ok

Code-blocks, delimited by curly-braces, also limit the scope of variables:

{ let a = 10; 
  let b = 20; 
  a + b }; 
//=> 30

A let expression can be supplied a "name". This name is called to repeatedly execute the let body.

let loop (i = 0) 
  if (i < 5) 
  { showln(i); 
    loop(i + 1) }; 
//-> 0
   1
   2
   3
   4

The case expression is used to compare the value of an expression with a number of clauses. The consequent part of the first matching clause is executed. If none of the clauses match, case will return void. Optionally an else part can be specified which will act as the default.

case (1 + 2) 
  1 -> 'one
| 2 -> 'two
| 3 -> 'three; 
//=> three

case (1 - 2) 
  1 -> 'one
| 2 -> 'two
| 3 -> 'three
| else -> 'none; 
//=> none

The match expression is similar to case but allow pattern matching and variable bindings using complex clauses:

match ([1, 2, 3]) 
  [1, a, b] -> a + b
| [2, a, b] -> a * b; 
//=> 5
match ([2, 2, 3]) 
  [1, a, b] -> a + b
| [2, a, b] -> a * b; 
//=> 6
match ( [3, 2, 3]) 
  [1, a, b] -> a + b
| [2, a, b] -> a * b
| else ->'none;
//=> none

record point(x, y); 
record three_d_point(x, y, z); 

function add(p) 
  | point(x, y) -> x + y
  | three_d_point(x, y, z) -> x + y + z; 

add(point(x = 20, y = 13.43)); 
//=> 33.43
add(three_d_point(x = 20, y = 13.43, z = 32.32)); 
//=> 65.75
