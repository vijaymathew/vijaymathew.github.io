Simple Expressions

Basic operations on numbers:

slogan> 3 + 4; 
=> 7
slogan> 3 + 4 * 10; 
=> 43
slogan> (3 + 4) * 10; 
=> 70
slogan> 10 / 4; 
=> 5/2
slogan> 10.0 / 4; 
=> 2.5
slogan> abs(-1); 
=> 1
slogan> sqrt(4.5); 
=> 2.1213203435596424

Define variables to "remember" values or results of computations:

slogan> define celsius = 30; 
slogan> define fahrenheit = 9 / 5 * celsius + 32; 
slogan> showln("The temperature is ", fahrenheit, " degrees Fahrenheit."); 
-> The temperature is 86 degrees Fahrenheit.

A variable can be assigned a new value:

slogan> celsius = 100; 
slogan> fahrenheit = 9 / 5 * celsius + 32; 
slogan> showln("The temperature is ", fahrenheit, " degrees Fahrenheit."); 
-> The temperature is 212 degrees Fahrenheit.

A couple of functions would have saved us the re-typing:

slogan> function convert(celsius) 9/5 * celsius + 32; 
slogan> function show_converted(celsius) 
          showln("The temperature is ", convert(celsius), " degrees Fahrenheit."); 

slogan> show_converted(30); 
-> The temperature is 86 degrees Fahrenheit.
slogan> show_converted(100); 
-> The temperature is 212 degrees Fahrenheit.
