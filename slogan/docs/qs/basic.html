Simple Expressions

Basic operations on numbers:

slogan> 3 + 4; 
=> 7
slogan> 3 + 4 * 10; 
=> 43
slogan> (3 + 4) * 10; 
=> 70
slogan> 10 / 4; 
=> 5/2
slogan> 10.0 / 4; 
=> 2.5
slogan> abs(-1); 
=> 1
slogan> sqrt(4.5); 
=> 2.1213203435596424

Define variables to "remember" values or results of computations:

slogan> define celsius = 30; 
slogan> define fahrenheit = 9 / 5 * celsius + 32; 
slogan> fahrenheit; 
=> 86

A variable can be assigned a new value:

slogan> celsius = 100; 
slogan> fahrenheit = 9 / 5 * celsius + 32; 
slogan> fahrenheit; 
=> 212

A couple of functions would have saved us the re-typing:

slogan> function convert(celsius) 9/5 * celsius + 32; 

slogan> convert(30); 
=> 86
slogan> convert(100); 
=> 212

A bunch of operations produce Boolean values:

slogan> 1 < 2; 
=> true
slogan> 1 * 20 >= 20; 
=> true
slogan> 20 == 2 * 5; 
=> false
slogan> not(20 == 2 * 5); 
=> true
slogan> true && false; 
=> false
slogan> true || false; 
=> true

slogan> define x = 10; 
slogan> and(x > 0, x <= 10, x/5 == 2); 
=> true
slogan> or(x > 0, x <= 10, x/5 == 3); 
=> true

Boolean values can be combined with if expressions to conditionally evaluate code:

slogan> if (10/2 == 5) "ok" else "blah!"; 
=> ok
slogan> if (10/2 == 50) "ok" else "blah!"; 
=> blah!

In fact, Slogan treats all values except the symbol false as truthy:

slogan> if (10/2) "ok" else "blah!"; 
=> ok
