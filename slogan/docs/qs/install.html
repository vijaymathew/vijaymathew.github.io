Installation

The following steps should work on GNU/Linux, OS X and other modern Unixes:

  1. Download and extract the Slogan source package.
  2. In the extracted folder, run the install command:
$ sudo ./install

Once the installation completes, you can run the Slogan interpreter:

$ slogan

This command will launch the REPL, where you can enter and evaluate programs. Let us try a couple of simple examples:

slogan> "hello, world"; 
=> hello, world

slogan> function greet(name) showln("hello, " name);
slogan> greet("Jack"); 
-> hello, Jack

Note: Throughout this document we use => to identify values produced by the interpreter and -> to identify values output by a function like showln.

The semicolon (;) asks the REPL to evaluate whatever we have entered so far. Terminating expressions and statements with a semicolon is optional in source code loaded/compiled from files.

You can get out of the REPL by calling the exit function:

slogan> exit();
