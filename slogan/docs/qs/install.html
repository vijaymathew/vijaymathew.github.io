Installation

The following steps should work on GNU/Linux, OS X and other modern Unixes:

  1. Download and extract the Slogan source zip file.
  2. In the extracted folder, run the usual build commands:
./configure
make

This will create the Slogan interpreter binary in the src sub-folder. Now you can run it by typing./src/slogan. If you want the interpreter to be available system-wide, run sudo make install.

When you run the Slogan interpreter, you will land in a REPL, where you can enter and evaluate programs. Let us try a couple of simple examples:

slogan> "hello, world"; 
=> hello, world

slogan> function greet(name) showln("hello, " name);
slogan> greet("Jack"); 
-> hello, Jack

Throughout this document we use => to identify values produced by the interpreter and -> to identify values output by a function like showln.

You can get out of the REPL by calling the exit function:

slogan> exit();
