FFI

FFI or the Foreign Function Interface is a facility that allow Slogan programs to call functions in shared libraries, which are usually written in the C programming language.

The following C program define two functions. The first function - add_i - expects two integers as arguments and return their sum. The second function - say_hello - consumes and returns a character string:

// demo_lib.c

#include <stdio.h>

int add_i(int a, int b)
{
  return a + b;
}

static char *message = "hi from C";
          
char *say_hello(char *s)
{
  printf("Slogan says: %s\n", s);
  return message;
}

The following command shows how to call the GNU C Compiler to convert this code into a shared library. 

gcc -Wall -shared -fPIC -o demo_lib.so demo_lib.c

On Mac OS X, the build-command will be:

gcc -dynamiclib -Wl,-undefined -Wl,dynamic_lookup -o demo_lib.so demo_lib.c

The declare ffi statement is used to load the library and define function prototypes required by Slogan:

declare ffi "demo_lib.so" [int add_i [int, int],
                           charstring say_hello [charstring]];

Now Slogan can call the C functions:

add_i(10, 20); 
//=> 30
say_hello("hi"); 
//-> Slogan says: hi
//=> hi from C
