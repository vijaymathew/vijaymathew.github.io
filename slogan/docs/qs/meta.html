Meta-Programming

The quote operator - % - is used to suspend the evaluation of an expression. The function eval can be called to evaluate a suspended expression during runtime:

slogan> define e = %10 * 20; 
slogan> e; 
=> [* 10 20]
slogan> eval(e); 
=> 200

A quoted expression can contain an unquote operator - %% - that can be used to insert the value of an expression into the suspended expression:

slogan> define e = let (i = 10) %10 * i; 
slogan> e; 
=> [* 10 i]
slogan> eval(e);
 error: Unbound variable: i
slogan> define e = let (i = 10) %10 * %%i; 
slogan> e; 
=> [* 10 10]
slogan> eval(e); 
=> 100

Macros are used for compile-time program generation. Here is a macro that defines a simple control structure: 

slogan> macro times(n, expr) 
         %let loop (c = %%n) 
          if (c > 0) { %%expr; 
                       loop(c - 1) }; 

slogan> times(3, showln("hello")); 
-> hello
   hello
   hello
