Exceptions

Examples:

function divide(a, b) 
  if (is_zero(b)) raise("Cannot divide by zero.") 
  else a / b; 

try divide(10, 2) 
catch (ex) showln(ex); 
//=> 5
try divide(10, 0) 
catch (ex) showln(ex); 
//-> Cannot divide by zero.

try raise(100) 
catch (ex) showln(ex) 
finally { showln("returning a default result..."); 0 }; 
//-> 100
//-> returning a default result...
//=> 0
