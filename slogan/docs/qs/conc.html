Concurrency

An expression can be evaluated concurrently in an independent task. The following program starts a task that will print a numeric sequence, outputting each number at an interval of two seconds:

slogan> task(let loop(i = 0) 
              if (i < 10) { showln(i); 
                            task_sleep(2); 
                            loop(i + 1) });

Two tasks can be synchronize and communicate using reactive variables:

slogan> define x = ?; 
slogan> task(showln(sqrt(?x)));

The new task will wait until a value is bound to the reactive variable x:

slogan> ?x = 10; 
-> 3.1622776601683795

Tasks can also communicate by message passing:

slogan> define sqrt_task = task(showln(sqrt(task_receive()))); 

slogan> task_send(sqrt_task, 10); 
-> 3.1622776601683795

A function object can be executed parallely on a separate processor core. Please see the chapter on Multitasking & Parallelism in the manual to learn more about this.
