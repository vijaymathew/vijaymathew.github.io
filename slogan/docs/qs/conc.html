Concurrency

The ! (task) operator evaluates an expression concurrently in an independent task. The following program starts a task that will print a numeric sequence, outputting each number at an interval of two seconds:

!let loop(i = 0) 
  when (i < 10)
  { showln(i); 
    task_sleep(2); 
    loop(i + 1) };

Two tasks can synchronize and communicate using reactive variables:

let x = ?; 
!showln(sqrt(?x));

The new task will wait until a value is bound to the reactive variable x:

?x = 10; 
//-> 3.1622776601683795

Tasks can also communicate by message passing. The !> operator sends a message to a task and the !< operator receives a message for the current task:

let sqrt_task = !showln(sqrt(!<)); 

sqrt_task !> 10; 
//-> 3.1622776601683795
