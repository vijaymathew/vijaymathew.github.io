Foreign Function Interface

The Slogan language offers a mechanism for interfacing Slogan code with C code. This mechanism is called the Foreign Function Interface or FFI. A Slogan program can call C functions exported by a shared (or dynamic-link) library. A C function to be callable from Slogan should meet the following conditions:

  1. all arguments should be one of two types: ___slogan_obj or void*.
  2. should accept a maximum of 8 arguments.
  3. the return type should be one of: void, void*, ___slogan_obj, char, char*, int, unsigned int, float or double.

The C type ___slogan_obj represents all Slogan objects (like strings, lists, integers and characters). The header file slogan.h defines the type ___slogan_obj along with functions and macros required for converting a ___slogan_obj to a C object and vice versa.

The chapter titled Interacting with the World in the tutorial gives a practical example on using the FFI.

Conversion Macros

This section describes the C marcos defined in slogan.h for converting between ___slogan_obj and C types.

C macro:___slogan_obj_to_int(s, t)
C macro:___slogan_obj_to_uint(s, t)
C macro:___slogan_obj_to_float(s, t)
C macro:___slogan_obj_to_double(s, t)
C macro:___slogan_obj_to_charstring(s, t)
C macro:___slogan_obj_to_nonnull_charstring(s, t)

The ___slogan_obj object s is converted to a C int, unsigned int, float, double or char* and stored in t which should be the address of a C variable of the appropriate type.

C macro:___int(s)
returns: a C int that contains the integer value of the Slogan character or 
         integer (___slogan_obj) s.
C macro:___fix(i)
returns: a ___slogan_obj containing the value of the C integer i.
C macro:___charstring_to_slogan_obj(c, s)
C macro:___nonnullcharstring_to_slogan_obj(c, s)

Initializes the ___slogan_obj s as a Slogan string with the characters from the C string c. ___charstring_to_slogan_obj creates a null-terminated string and ___nonnullcharstring_to_slogan_obj creates a string object that keeps track of its length. The C string c should always be null terminated.

The following C code shows the correct way to use these functions:

___slogan_obj hi()
{
  ___slogan_obj message;
  char *s = "hi from C!";
  ___nonnullcharstring_to_slogan_obj(s, &message);
  ___release_slogan_obj(message);
  return message;
}

The function hi can be invoked from Slogan as shown below, assuming that it has been compiled into a shared library called "test.so":

slogan> define clib = ffi_open("./test.so"); 
slogan> ffi_call_obj_0(clib, "hi"); 
=> hi from C!
slogan> ffi_close(clib); 
=> true
C macro:___pair(obj1, obj2)
returns: a ___slogan_obj that represents a Slogan pair with head and tail 
         pointing to obj1 and obj2.

Both obj1 and obj2 must be of types ___slogan_obj.

C macro:___head(p)
C macro:___tail(p)
returns: the head or tail objects of the ___slogan_obj p that represents a Slogan pair.
C macro:___head(p)
C macro:___tail(p)
returns: the head or tail objects of the ___slogan_obj p that represents a Slogan pair.
C constant:___TRU(p)
C constant:___FAL(p)
returns: the ___slogan_obj that represents the Slogan values true or false.
C constant:___NUL(p)
returns: the ___slogan_obj that represents the empty list [].
C macro:___is_empty(pair)
returns: ___TRU if pair is the empty list [], ___FAL otherwise.

The following C function demonstrates how to receive a list of integers from Slogan and print it:

void print_i_list(___slogan_obj a)
{
  while (!___is_empty(a)) {
    ___slogan_obj h = ___head(a);
    ___slogan_obj t = ___tail(a);
    int i;
    ___slogan_obj_to_int(h, &i);
    printf("%d\n", i);
    a = t;
  }
}

Print_i_list can be invoked from Slogan as:

slogan> ffi_call_void_1(clib, "print_i_list", [1, 10, -3, 99]); 
-> 1
   10
   -3
   99
C macro:___body(array)
returns: a C array of ___slogan_obj objects that are the elements of the Slogan array array.

The following C function prints the first two elements of a Slogan array passed to it. The function expects the first element to be an integer and the second element to be a string.

void f(___slogan_obj a)
{
  ___slogan_obj *b = ___body(a);
  int i;
  char *s;
  ___slogan_obj_to_int(b[0], &i);
  ___slogan_obj_to_charstring(b[1], &s);
  printf("%d\n%s\n", i, s);
}

This function can be invoked from Slogan as:

slogan> ffi_call_void_1(clib, "f", #[100, "hello"]); 
-> 100
   hello
