Introduction

Slogan is a programming language designed for concurrent, networked, high-performance applications. It features first-class functions, lexical scoping and operations on structured data like strings, lists, arrays, sets and hashtables. It allow users to dynamically create any number of concurrent tasks and to take full advantage of modern mult-core computers. Its powerful data, control and syntactic abstraction capabilities makes Slogan a language that can be extended to the core.

This book is designed to help the reader learn how to program in Slogan. It is assumed that you are already familiar with one or more languages like C, Java or Python.

We start with a tutorial and each major feature of the language is explained in subsequent chapters. The last part of the book is a complete reference to the syntax of the language and its built-in functions.

Please visit the Slogan website to download and install Slogan.
