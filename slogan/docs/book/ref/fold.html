fold


fold(fn, init, @rest seq, ...)

Return init if the sequences seq, ... are all empty. If they are not empty, apply the function fn to init and the heads of seq, ..., then recur with the value returned by fn in place of init and the tails of seq, ....

Return the final value of fn.

The sequence arguments should all have the same length. Fn should accept one more argument than the number of sequences and return a single value. It should not mutate the sequences.

Examples:


fold(`-`, 0, [1, 2, 3, 4])
// -10
