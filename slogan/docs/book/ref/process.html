process


process(fn, @optional timeout)

Arranges the function fn to be executed parallely. If timeout is specified, an error is raised if the spawned process does not respond before timeout is reached. Return an object that contain information about the newly started process.

If provided, timeout should be an exact or inexact real number representing a relative time in seconds from the moment process was called.

Examples:


let p = process(^(cp) showln("server says: ", process_receive(cp)))
process_send(p, "hello")
//> server says: hello
process_close(p)
