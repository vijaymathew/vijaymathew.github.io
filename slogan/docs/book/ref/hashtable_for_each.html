hashtable_for_each


hashtable_for_each(fn, ht)

For each entry in the hash table ht, call the function fn by passing it the key and value as arguments.

Return void.

Examples:


let h = make_eq_hashtable()
hashtable_set(h, 'name, "nemo")
hashtable_set(h, 'age, 1)
hashtable_for_each(^(k, v) println(k, ": ", v), h)
//> name: nemo
    age: 1
