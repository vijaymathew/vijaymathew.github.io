Programming Slogan

This is a tutorial on the Slogan programming language. It also serves as a gentle introduction to the principles of functional programming, concurrency and fault-tolerant distributed computing. This is not intended to be a general programming tutorial. It is assumed that the reader have some programming experience and is familiar with terms commonly associated with computers and programming languages.
