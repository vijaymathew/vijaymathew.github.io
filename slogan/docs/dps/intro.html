Introduction

This chapter gives you a gentle, hands-on introduction to the most important concepts covered in this tutorial. Later chapters contain more in-depth coverage.

Simple arithmetic

Let us start by using Slogan as a simple calculator. Start the Slogan interpreter by typing:

slogan

You should now see a prompt like this:

slogan>

You can enter arithmetic expressions at the prompt and Slogan will respond with their results:

slogan> 3 + 4;
=> 7
slogan> 10_000 * 99_000 + 1_89_0000;
=> 991890000

As you can see from the last expression, underscores can be added to numbers to improve readability.

Variables

We would like to preserve the result of a calculation so that we can reuse it later. This can be achieved by defining a variable and assigning it to the result: 

slogan> define x = 3 + 4;
slogan> x ;
=> 7
slogan> x * 100;
=> 700
slogan> x = 10; 
slogan> x * 100; 
=> 1000

Variables are just shortcuts for values. You can assign a new value to a variable or you can define another variable with the same name as a previous one. The previous variable then becomes inaccessible. This does not mean that the value that the variable referred to also disappears from the system. Other variables could still point to it.

A variable identifier must start with an alphabet or one of these special characters: '_', '$', '?', '~', '@' This can be followed by any number of characters or digits. Some examples of valid variable identifiers are: x, xyz, x123, hello_world, and $hello@world123.

Note that identifiers starting with '?' and '@' are specially interpreted in certain contexts. These special contexts will be discussed in later chapters.
