Introduction

This chapter gives you a gentle, hands-on introduction to the most important concepts covered in this tutorial. Later chapters contain more in-depth coverage.

Simple arithmetic

Let us start by using Slogan as a simple calculator. Start the Slogan interpreter by typing:

slogan

You should now see a prompt like this:

slogan>

You can enter arithmetic expressions at the prompt and Slogan will respond with their results:


slogan> 3 + 4;
=> 7
slogan> 10_000 * 99_000 + 1_89_0000;
=> 991890000

As you can see from the last expression, underscores can be added to numbers to improve readability.
