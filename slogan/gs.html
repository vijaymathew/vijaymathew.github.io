Getting started with Slogan

Obtaining and running Slogan

The latest development code base of Slogan can be obtained from GitHub:

$ git clone https://github.com/vijaymathew/slogan.git

The repository will be cloned to a directory named slogan. Use these commands to compile the code:

$ ./configure-platform
$ make
$ sudo make install
$ slogan

The last command will launch the Slogan REPL where you will see a prompt like this:

slogan>

The REPL is where you enter Slogan program expressions and evaluate them to see results.

(If you don't want to install Slogan, after `make', type `./src/slogan' to launch the REPL.)
